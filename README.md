# 🧑‍💻 Моё портфолио

Одностраничный сайт-портфолио с тёмной/светлой темой, анимированным интерфейсом и интерактивными разделами.

🌐 **Демо:** [shvcxz.github.io/portfolio16](https://shvcxz.github.io/portfolio16/)

![image](https://i.imgur.com/pePbMuj.jpeg)

---

## ✨ Возможности

- 🎨 **Переключатель тем** – тёмная, светлая, тёмно-синяя и случайная генерация цветовых схем
- 🧩 **Интерактивные разделы**:
  - *Мои проекты*: Weather Dashboard, ИИ-музыка (Suno AI), Тетрис
  - *Мои услуги*: веб-разработка, адаптивная вёрстка, Fortnite коучинг, VALORANT буст
  - *Хобби и увлечения*
  - *Контакты*: Telegram, Discord, GitHub, YouTube, Email с копированием в буфер
- 🧭 **Фиксированная навигация** с бургер-меню для мобильных
- ✨ **Анимации появления** элементов при скролле
- 💡 **Hover‑эффекты** с фиолетовым свечением иконок
- 📋 **Копирование email** в один клик с визуальным уведомлением
- 📱 **Полностью адаптивный дизайн**

---

## 🛠️ Технологии

- HTML5 (семантическая вёрстка)
- CSS3 (кастомные свойства, Flexbox/Grid, переходы, анимации)
- JavaScript (ES6+): управление темами, бургер-меню, копирование, scroll-анимации
- [Font Awesome 6](https://fontawesome.com/) – иконки
